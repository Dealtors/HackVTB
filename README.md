# Оркестр API

**Интеллектуальная платформа для автоматического тестирования бизнес-процессов**

## Проблема → Решение

**Проблема:** Тестировщики и аналитики тратят значительное количество времени на ручное сопоставление BPMN-диаграмм и других диаграмм с API-эндпоинтами и создание тестовых данных.

**Решение:** Оркестр API автоматически анализирует BPMN и другие и OpenAPI-спецификации, строит сквозные тестовые сценарии и генерирует данные с помощью ИИ.

Помимо выполнения основного функционала, планируется на энд-поинтах сделать карточки, в котором можно редактировать текст, а также присутствовать информация об энд-поинтах API и основные данные, которые будут нужны. Тестовый ввод сохранения данных. Функция self-healing, автоматический перезапуск тестов с альтернативным вариантов, если "упадут".

## Быстрый старт

### Требования
- Java 17+
- PostgreSQL 14+
- Node.js 18+
- Maven 3.8+

### Запуск проекта

1. **Клонируйте репозиторий**
```bash
git clone https://github.com/your-org/orchestra-api.git
cd orchestra-api
```
2. **Настройте базу данных**
```sql
CREATE DATABASE orchestra_db;
CREATE USER orchestra_user WITH PASSWORD 'password';
```
3. Запустите бэкенд
```bash
cd backend
mvn spring-boot:run
```
4. Запустите бэкенд
```bash
cd frontend
npm install
npm start
```

5. Откройте в браузере
```
http://localhost:3000
```

## Демо-сценарий
1. Загрузите BPMN-диаграмму процесса; 
2. Загрузите OpenAPI-спецификацию вашего API;
3. Нажмите "Run Test" для автоматического тестирования;
4. Наблюдайте выполнение каждого шага в реальном времени;
5. Скачайте отчет с детальными логами и аналитикой.
